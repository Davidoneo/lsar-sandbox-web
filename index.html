Bypassing Windows Execution Restrictions

As an independently distributed tool without a commercial EV certificate, Windows SmartScreen may flag the executable. To run the app:

  • Browser (Edge/Chrome): Click the downloads icon, select the three dots next to the file, choose "Keep", and then "Keep anyway".
  • SmartScreen: When double-clicking the `.exe`, click "More info" in the upper left, then "Run anyway".
  • Windows 11: Right-click the `.exe` file, select Properties, check the "Unblock" box at the bottom of the General tab, and click Apply.

Bridging the Logistics Gap

Enterprise LSAR software often requires expensive licenses and complex IT infrastructure. Minimal LSAR Sandbox eliminates these barriers by offering a free, lightweight alternative designed for:

  • Academia & Training: Learn the data schema mechanics of MIL-STD-1388-2B.
  • SMEs: Manage smaller defense subcontracts without massive corporate platform overhead.
  • ILS/IPS Engineers: A local test sandbox to parse and validate positional flat-files from sub-vendors prior to enterprise database imports.

Technical Objectives

Zero Server Setup

No installers or DB admins required. The SQLite engine is fully embedded in the local executable. Work locally with 100% data privacy.

Strict Data Integrity

Actively manages relational foreign key structures. It prevents broken logistics trees by rejecting entries with orphaned LCN relations or invalid references.

Positional Text Exchange

Import and export standard fixed-width positional flat files (.txt). Includes interactive error-tracking to highlight alignment and syntax conflicts.

Interface Overview

Logistics Structure Definition (Table XB)

Hierarchical visualization and structural management of the physical and functional breakdown tree.

Component Catalog (Table HA)

Direct relational mapping of manufacturer part numbers and CAGE codes prior to provisioning.

Fixed-Width Export Processing

Granular export manager engineered to compute character paddings and compile compliant flat files (.txt format).
