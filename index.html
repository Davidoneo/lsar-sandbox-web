Minimal LSAR Sandbox

A standalone, zero-setup MIL-STD-1388-2B relational database engine.

Engineering-Grade Logistics Data, simplified.

Minimal LSAR Sandbox is a lightweight, portable Windows application designed for Integrated Logistic Support (ILS) professionals. It allows you to explore, edit, and validate MIL-STD-1388-2B data structures without complex server setups or legacy software dependencies.

📦 Zero Installation

Delivered as a single executable file. No admin rights required, no database servers to configure. Just run it.

🔗 Relational Integrity

Built on a strict SQLite schema. It actively enforces foreign keys and structural dependencies (A, C, E, H tables, etc.) exactly as the standard dictates.

📄 Flat-File Support

Import and export standard fixed-width `.txt` positional files effortlessly. Perfect for validating vendor data or training new engineers.
